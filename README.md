# Task Board

Small full-stack task board project with Spring Boot, React, WebSocket updates, and CI.

## Stack
- Java 17 + Spring Boot
- React + TypeScript + Vite
- H2 (local) / PostgreSQL (compose)
- GitHub Actions

## Local run
- Backend: `cd backend && mvn spring-boot:run`
- Frontend: `cd frontend && npm install && npm run dev`
